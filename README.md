# JLPT Hijack

한국어 사이트를 읽으며, 한국 사이트의 내용을 JLPT 단어와 매칭시켜 학습을 도와주는 역할의 익스텐션.

## 왜 만들었나

공부하려고 하는데 평소에 일본어 공부방법이 별로 없더라.

안키 앱 또한 결국 시간을 내어 핸드폰을 열고, 단어를 지속적으로 외우는 방법이더라.

다른 익스텐션을 찾아봤는데, 일본어 사이트에서 바뀌거나 팝업하는거밖에 없어서 평소에 보기 힘들더라.

이 컨셉의 앱은 시장에 없더라.

강제적으로 바꿔주는 것도 없더라구.

그래서 내가 만들기로 했어.


## 3가지 표시 모드

- **밑줄** (기본) — 매칭 단어에 밑줄만 표시, 호버 시 일본어 팝업
- **루비** — 한국어 옆에 일본어를 작게 병기
- **강제모드** — 한국어를 일본어로 완전히 대체

N5~N1 체크박스로 원하는 레벨만 필터링 가능.

## 기술 스택

- Chrome Extension Manifest V3 (Vite + CRXJS)
- TypeScript + React (팝업 UI)
- IndexedDB (사전 데이터 저장)
- Service Worker (역방향 인덱스 + LRU 캐시)
- 서버 없음 — 전부 클라이언트에서 동작

## 사전 데이터

JMdict(CC-BY-SA) 기반 22,578개 엔트리에 JLPT 레벨 태그(N1~N5)를 매핑하고, 일본어→한국어 번역을 자동화 파이프라인으로 생성했다.

| 항목 | 수치 |
|------|------|
| 전체 엔트리 | 22,578개 |
| 한국어 번역 | 21,975개 (97.3%) |
| JLPT 태그 | N1: 2,974 / N2: 1,548 / N3: 1,585 / N4: 541 / N5: 560 |

### 번역 파이프라인

1. `scripts/build-dict.ts` — JMdict 다운로드 + JLPT 태그 매핑 + 한국어 캐시 적용
2. `scripts/translate-ko.ts` — Claude CLI로 일본어→한국어 배치 번역
3. `scripts/verify-translations.ts` — 오역 검증 (겹치는 한국어, 한자음 불일치)
4. `scripts/fix-overlaps.ts` — 겹치는 한국어를 그룹으로 묶어 재번역

## 사용 방법

1. 한국어 사이트 아무거나 열면 자동으로 JLPT 단어 감지
2. 익스텐션 아이콘 클릭 → 표시 모드(밑줄/루비/강제) 전환
3. 레벨 필터(N5~N1)로 원하는 레벨만 선택
