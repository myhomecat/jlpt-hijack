JLPT Hijack - 개인정보처리방침

최종 수정일: 2026-04-02

개인정보 수집

JLPT Hijack은 개인정보를 수집, 저장, 전송하지 않습니다. 모든 처리는 브라우저 내에서 로컬로 이루어집니다.

권한

이 익스텐션은 한국어 웹페이지의 텍스트를 스캔하여 JLPT 단어와 매칭하기 위해 모든 URL에 대한 접근 권한을 요청합니다. 외부 서버로 데이터를 전송하지 않습니다.

저장

사전 데이터와 사용자 설정(표시 모드, 레벨 필터)은 Chrome Storage와 IndexedDB를 사용하여 로컬에 저장됩니다. 이 데이터는 기기 밖으로 나가지 않습니다.

문의

문의사항은 github.com/myhomecat/jlpt-hijack에 이슈를 등록해주세요.
