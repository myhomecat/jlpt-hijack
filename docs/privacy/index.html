JLPT Hijack - Privacy Policy

Last updated: 2026-04-02

Data Collection

JLPT Hijack does not collect, store, or transmit any personal data. All processing happens locally in your browser.

Permissions

This extension requests access to all URLs solely to scan Korean text on web pages and match JLPT vocabulary. No data is sent to any external server.

Storage

Dictionary data and user settings (display mode, level filter) are stored locally using Chrome Storage and IndexedDB. This data never leaves your device.

Contact

Questions? Open an issue at github.com/myhomecat/jlpt-hijack
